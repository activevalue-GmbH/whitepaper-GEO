# active value Whitepaper GEO

Statischer HTML-Starter fuer ein laengeres Whitepaper im active-value-Stil mit Fokus auf:

- mobile Lesbarkeit
- gut funktionierende Tabellen und Uebersichten
- mehrfach platzierte Call-to-Actions
- einfache Domain-Anfrage per vorbereiteter E-Mail

## Dateien

- `index.html`: kompletter Onepager
- `README.md`: Kurzbeschreibung

## Lokal ansehen

`index.html` direkt im Browser oeffnen.

## Offene Punkte fuer die Finalisierung

- echtes Whitepaper-Thema und finaler Inhalt
- finale Empfaengeradresse fuer Anton
- Deployment-Ziel, z. B. GitLab Pages

## CTA-Logik

Die Domain-Check-CTA ist bewusst dreifach angelegt:

1. direkt im Hero fuer sehr entschlossene Nutzer:innen
2. noch einmal im Mittelteil nach dem Reifegrad-Abschnitt
3. ein drittes Mal im Abschlussbereich
